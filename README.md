# Smart File Organizer

## Objective

Automate file organization using Python.

## Features

* Organizes files based on type
* Uses OS Module
* Uses Exception Handling
* Generates Logs
* Supports User Input

## Technologies Used

* Python
* OS Module
* shutil Module
* logging Module

## How to Run

```bash
python file_organizer.py
```

Enter the folder path when prompted.

## Output

Files are automatically sorted into folders:

* Images
* Documents
* Audio
* Videos
* Archives
